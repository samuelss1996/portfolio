The Global Game Jam 2020 took place between January 31st and February 2nd. It was held worldwide and I participated from Gothenburg, Sweden at Studio Level Up.

The theme of the Game Jam was repair.

Accordingly, I developed Project Daemon, a game in which the main character is a computer daemon that has to repair damaged sectors of the hard disk.

You can read more about my game, Project Daemon here.

In the end, I was satisfied with the result. It was declared the local winner of the site from which I participated, with other 17 videogames being uploaded.
