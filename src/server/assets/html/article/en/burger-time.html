Introduction

This is a re-implementation of BurgerTime, a 1982 arcade game. The goal of this game is to complete a number of burgers while avoiding enemy foods.

Development

This project was developed as a part of one of my Master's courses: Game Engine Architecture.

The game was re-implemented using plain C++, with no third-party game engines. Instead, a basic game engine architecture was designed and implemented from scratch.

The only third-party library used was SDL2, that simply allows to easily draw sprites on the screen and play sound effects/music.
