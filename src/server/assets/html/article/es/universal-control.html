Introduction

Universal Control es un juego en 2.5D en el que los jugadores toman control de una mano gigante para proteger al planeta Tierra de un meteorito enorme que se aproxima. El jugador tiene que lanzar el meteorito hacia el agujero negro para completar el puzzle y avanzar al siguiente nivel. Sin embargo, el meteorito se encontrará en su camino diferentes obstáculos tales como planetas o estrellas con gravedad propia, que desviarán su trayectoria.

Desarrollo

This game was developed in roughly two weeks as a part of one of my Master's courses.

Este juego fue desarrollado en dos semanas como parte de una asignatura de mi máster.

El motor usado fue Unity. El principal desafío fue mostrar la predicción de la trayectoria como una línea de puntos. Para conseguirlo, una versión simplicada de las físicas del RigidBody de Unity fue implementada usando integración Verlet.
