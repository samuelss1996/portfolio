Introducción

Project Daemon es el juego que desarrollé para la Global Game Jam 2020.

En Project Daemon el jugador controla un demonio de un sistema operativo.

En computación, un demonio es un proceso que se ejecuta de fondo, gestionado por el sistema operativo, ejecutando tareas que el usuario no ve, pero que son necesarias para el correcto funcionamiento del sistema.

El demonio que controla el jugador se encarga the arreglar sectores dañados del disco duro, siguiendo las órdenadas dadas por "Kernel", el personaje que gobierna todo el lugar. Para arreglar estos sectores, el jugador debe alcanzar las cajas rojas final de cada nivel, convirtiéndolas así en verde.

Esto está inspirado por los programas de análisis de discos duros del mundo real, que habitualmente muestran los sectores del disco duro en una tabla en la cual los sectores correctos están marcados como una celda en verde y los incorrectos como una celda en rojo. Desde el punto de vista de la jugabilidad, Project Daemon es principalmente juego de plataformas en 2D, con algunas influencias de juegos basados en ritmo..

Desarrollo

El juego fue desarrollado usando Unity, durante las 48 horas de la Global Game Jam 2020.
