Introduction

Eggsquisite is a puzzle game developed during the Nordic Game Jam 2019 hosted in Copenhagen, Denmark. It was designed with according to the game jam theme: "that again".

Development

The game was developed during the weekend the game jam took place. Unity was used as the game engine.
